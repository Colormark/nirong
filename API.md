# References
## [PlainObject](http://api.jquery.com/Types/#PlainObject)
PlainObject 类型是一个包含零个或多个键值对的 JavaScript 对象。  
jQuery中的isPlainObject() 函数用于判断指定参数是否是一个纯粹的对象。
